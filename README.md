#### Aplikacja w zamyśle pozwalająca na sterowanie urządzeniami. Zostawiona na etapie dodawania kart i zapisywania ich w async storage.

##### Aplikacja posiada splash screen.

![splash](https://drive.google.com/uc?id=1D7snTtdQ0OvJ_Z9K2WQlnTYoC-xH_ERe)

##### Po uruchomieniu widoczny jest ekran główny. Jeśli jest to pierwsze uruchomienie aplikacji na ekranie znajdzie się tylko przycisk dodania urządzenia.

![home](https://drive.google.com/uc?id=1oRQ-wNeng7Q64VkPGjZRfp9GxEduaS4Y)
